# Skills

Los skills del proyecto viven en `/.github/skills/` (raíz del repo) para ser descubiertos automáticamente por VS Code Copilot.

## Skills disponibles

| Skill               | Invocación           | Propósito                                                      |
| ------------------- | -------------------- | -------------------------------------------------------------- |
| `react-component`   | `/react-component`   | Crear componentes UI con shadcn/ui + CVA + Tailwind CSS 4.1    |
| `feature-page`      | `/feature-page`      | Crear páginas completas con routing, layout, guards, SEO, i18n |
| `admin-crud`        | `/admin-crud`        | Construir páginas CRUD del panel admin con tablas, formularios |
| `appwrite-function` | `/appwrite-function` | Crear Appwrite Cloud Functions con validación y roles          |
| `appwrite-schema`   | `/appwrite-schema`   | Gestionar colecciones, atributos, índices y sync CLI           |
| `service-hook`      | `/service-hook`      | Crear servicios mock y hooks React Query                       |
| `i18n-content`      | `/i18n-content`      | Agregar o actualizar traducciones ES/EN                        |
| `mock-data`         | `/mock-data`         | Crear o actualizar datos mock centralizados                    |
