# 00_PROJECT_BRIEF - YOGA WELLNESS KITCHEN PLATFORM

## Visión

Construir una plataforma moderna para vender una experiencia integral de bienestar:
- clases de yoga,
- membresías,
- paquetes,
- complementos wellness kitchen.

## Objetivo MVP

- landing comercial premium,
- exploración pública de clases,
- detalle de clase,
- flujo de reserva/compra,
- checkout,
- dashboard de cliente,
- dashboard admin,
- gestión de clases, horarios, membresías, paquetes, productos, leads, ventas y clientes.

## Diferencial

- combina yoga + wellness kitchen;
- vende servicios y complementos dentro del mismo flujo;
- experiencia visual premium y moderna;
- mobile-first;
- preparada para i18n y Appwrite desde el inicio.
